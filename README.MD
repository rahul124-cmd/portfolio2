# Rahul Gopakumar - Personal Portfolio

A modern, responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. The portfolio showcases my academic background, technical skills, leadership experience, achievements, and contact information in a clean and professional interface.

---

## Preview

The portfolio features:

- Dark themed modern UI
- Responsive layout
- Animated typing effect
- Education timeline
- Skills section
- Coursework section
- Leadership experience
- Achievements
- Resume download
- Contact form
- Scroll progress indicator
- Back-to-top button
- Smooth scrolling animations

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter & JetBrains Mono)

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Open the project folder

```bash
cd portfolio
```

Run the project

Simply open

```
index.html
```

in your preferred web browser.

No additional dependencies or installation are required.

## Customization

### Change Profile Picture

Replace

```
images/profile-placeholder.png
```

with your own image.

---

### Change Background Image

Replace

```
images/background-placeholder.jpg
```

with your preferred background.

---

### Update Resume

Replace

```
resume.pdf
```

with your latest resume.

---

### Update Contact Details

Edit the following inside

```
index.html
```

- Name
- Email
- Phone Number
- LinkedIn
- GitHub
